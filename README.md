# 单片机模板库

#### 这是什么

这是基于 51 单片机语音时钟的代码


### 命名规则

宏定义标识符采用全大写
> #define **RS_CLR** RS=0

> #define **RS_SET** RS=1

外围芯片、模块、传感器等采用大写，如
> DS18B20、DS1302、LCD1602、WT588F

函数名采用大驼峰命名(兴)

> TimerInit();

> DS1320InIt();

> DS1302ReadBit();

> DelyaUs();


变量名采用小驼峰命名


### 函数返回值代表含义

| 返回值 | 函数执行情况 | 返回数据 |
| :----: | :----------: | :------: |
|   0    |   执行成功   |  数据 0  |
|   1    |              |  数据 1  |
|   -1   |   执行错误   |          |
|  其他  |              |   其他   |
